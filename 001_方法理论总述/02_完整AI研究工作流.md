## 二、选题阶段详细操作

### Step 1：用 Elicit 搜索现有研究

访问 elicit.org，搜索你的核心关键词组合：
- `[研究对象] + [研究方法] + [问题类型]`
- 导出近 3 年文献列表，观察发表趋势

### Step 2：用 Claude 分析研究空白

```
Prompt 模板：
你是[领域]方向的资深研究者。

研究方向：[填入你的方向]

请帮我：
1. 梳理这个领域目前已有的主要研究方向（3-5个）
2. 识别尚未被充分研究的空白（具体、可操作）
3. 推荐 3 个有发表价值的具体研究切入点，说明理由
```

### Step 3：你来决策

考量维度：
- 导师研究方向的契合度
- 是否有数据/样本获取渠道
- 目标期刊的偏好类型
- 预期研究周期是否匹配

## 三、调研阶段详细操作

### Step 1：批量获取文献

- **Semantic Scholar**：semanticscholar.org，支持语义搜索
- **Elicit**：自动摘要，快速筛选
- **Connected Papers**：可视化引用网络，发现被遗漏的核心文献
- **Zotero**：统一管理，自动生成引用格式

### Step 2：NotebookLM 问答式理解

上传 8-10 篇核心文献后，依次提问：

```
问题序列：
1. 综合这些论文，领域内仍未解决的核心问题是什么？
2. 这些研究在方法论上有哪些共同局限？
3. 如果我要做一篇新研究，最有价值的切入点在哪里？
4. [具体论文标题] 与其他论文相比，最大创新点和局限是什么？
```

### Step 3：生成文献综述框架

```
Prompt 模板：
基于以下研究主题：[填入]

请帮我生成文献综述框架：
- 4 个小节，每节有明确的综述逻辑
- 每节应覆盖哪类文献
- 每节结尾引出的"研究空白"是什么
```

## 四、写作阶段详细操作

### 摘要生成

```
Prompt 模板：
研究主题：[填入]
核心论点：[填入]
主要发现：[填入具体数据或结论]
研究方法：[填入]

请生成符合[目标期刊类型]标准的英文摘要，150-200词。
结构：背景（1-2句）/ 方法（2-3句）/ 结果（2-3句）/ 结论（1-2句）
```

### Introduction 逻辑设计

```
Prompt 模板：
请为上述研究设计 Introduction 的论证逻辑：
- 漏斗结构：宏观背景 → 具体问题 → 研究空白 → 本文贡献
- 每段核心论点一句话概括
- 每段需要引用的文献类型
```

### AI 模拟 Reviewer 预审

```
Prompt 模板：
以下是我的论文摘要和框架：[粘贴内容]

请扮演[目标期刊名]的审稿人，列出：
1. 你会提出的 3 个主要质疑（苛刻、具体）
2. 方法论上的潜在漏洞
3. 你认为最需要补强的部分
```

## 五、投稿与修改阶段

### 期刊推荐

```
Prompt 模板：
论文主题：[简述]
主要贡献：[简述]
方法类型：[定量/定性/混合]
目标读者：[学术/政策/业界]

推荐 5 个适合投稿的期刊：
期刊名称、影响因子区间、接收该类研究的频率、审稿周期
```

### Response Letter 起草

```
Prompt 模板：
以下是审稿人意见：[粘贴]

请起草回复信框架：
1. 开场致谢（1-2句，专业不卑微）
2. 每条意见的回应逻辑（接受/有条件接受/礼貌拒绝）
3. 说明我们做了哪些修改
语气：自信、专业、建设性
```

## 六、建立个人 SOP

### SOP 文档结构

```
我的论文生产 SOP（版本：x.x）

选题阶段：
- 使用工具：___
- 核心 Prompt：___
- 质量检查标准：___

[依此类推每个阶段]

上次更新：___
更新原因：___
```

### 反馈环数据来源

- **拒稿信**：分析 Reviewer 意见类型，记录高频问题
- **接受信**：分析哪些要素有效，下次强化
- **每篇复盘**：写一段"这次流程什么最有效/最浪费时间"
