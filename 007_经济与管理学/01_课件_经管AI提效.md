# 经济与管理学方向：AI 赋能研究提效

## 一、本方向研究特点与 AI 适配性分析

| 研究特点 | AI 适配程度 | 说明 |
|---------|-----------|------|
| 计量经济学分析 | 高 | 代码生成（Stata/R/Python）|
| 文献综述 | 高 | 与通用方法一致 |
| 问卷设计与量表开发 | 中高 | AI 辅助条目生成，人做效度判断 |
| 案例研究与访谈 | 中高 | 编码辅助 |
| 数学模型推导 | 中 | AI 辅助思路，核心推导需人完成 |
| 政策报告写作 | 高 | 结构化写作 AI 非常有效 |

**经管研究者的时间黑洞**：
- 计量分析代码调试（Stata/R 报错）
- 文献综述的大量阅读
- 问卷预处理和数据清洗
- 英文写作（尤其是 Introduction 和结论）

## 三、AI 工作流：实证经济学方向

```
确定研究问题和因果识别策略
    ↓
Elicit / Google Scholar 检索相关文献
NotebookLM 上传核心论文 → 理解方法论细节
    ↓
Claude 分析现有文献的数据来源和识别策略
[你] 确定自己的研究设计（核心创新）
    ↓
数据获取（WRDS / Wind / 官方统计数据）
Cursor 生成数据清洗和描述性统计代码
[你] 确认数据处理逻辑
    ↓
Cursor 生成主回归代码（OLS / IV / DID / RDD 等）
[你] 判断模型设定的经济学含义
Claude 辅助结果解读初稿
    ↓
Claude 起草论文，[你] 修改经济学逻辑
```

## 四、实战案例：双重差分（DID）政策评估研究

### 场景：评估某政策对企业绩效的影响

**研究设计阶段**：

**Prompt**：
```
我想研究[某政策]对[企业绩效指标]的影响
使用双重差分方法
实验组：[政策覆盖的企业]
控制组：[未受政策影响的企业]
数据：[上市公司年报数据，2015-2022年]

请帮我：
1. 梳理 DID 方法的识别假设，以及我如何验证它们
2. 建议主要的稳健性检验清单
3. 潜在的内生性问题和处理思路
```

---

### 代码生成（Stata）

**Prompt（Cursor 中输入）**：
```
我需要用 Stata 实现以下分析：
数据格式：面板数据，变量包括 id（企业代码）、year、
          treat（是否为处理组）、post（政策后）、
          y（被解释变量，ROA）、controls（控制变量列表）

需要：
1. 描述性统计（分处理组/控制组）
2. 平行趋势检验（事件研究图）
3. 主回归（DID + 控制变量 + 固定效应 + 聚类标准误）
4. 安慰剂检验（随机化政策时点）
5. PSM-DID 稳健性检验
输出所有结果到 Word 格式表格
```

**Cursor 输出**：完整 Stata do 文件（约 150 行）

---

### 结果解读辅助

**Prompt**：
```
以下是我的 DID 主回归结果：
[粘贴回归系数、标准误、显著性]

请帮我：
1. 用经济学语言解释系数的含义（数量级和方向）
2. 与文献中类似研究的结果对比（我提供的参考文献：[列出]）
3. 起草 Results 部分的文字描述（学术英文）
4. 指出审稿人可能质疑的统计问题
```

## 五、管理学研究专项：问卷与量表

### 问卷条目生成

```
我在研究[变量名，如：员工创新行为]
已有的成熟量表：[列出]

请：
1. 综合现有量表，推荐适合我研究情境的条目
2. 针对[中国情境/互联网行业]的特殊性，建议修改哪些条目
3. 生成 3-4 个本地化的新条目，符合7点李克特量表格式
注意：请标注每个条目来源，我需要查阅原文确认
```

### 信效度分析代码（R）

```
我有问卷数据（SPSS格式），需要：
1. 导入数据，检查缺失值
2. 各量表的 Cronbach's α
3. 验证性因子分析（CFA），输出拟合指标（CFI/TLI/RMSEA）
4. AVE 和 CR 计算（收敛效度）
5. HTMT 检验（区分效度）
用 R 实现（lavaan + semTools 包）
```

## 六、经管英文写作专项技巧

### Introduction 的"漏斗结构"

```
我的研究：
- 大背景：[宏观现象或政策背景]
- 具体问题：[企业/个人层面的具体问题]
- 研究空白：[现有文献没有解决什么]
- 本文贡献：[我的研究如何填补空白]

请起草 Introduction（约600词），
符合 AMJ / Strategic Management Journal 的写作风格，
每段清晰承接，不要用模板化措辞
```

### 经济学论文 Conclusion

```
以下是我的主要发现：[描述]
以下是我的理论贡献：[描述]
以下是政策含义：[描述]
研究局限：[描述]

请起草 Conclusion 部分（约400词），
避免仅仅重复 Abstract，需要提升到更高层次的意义讨论
```
