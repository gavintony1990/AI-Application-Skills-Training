# 理工科方向：AI 赋能研究提效

---

## 核心论点

> 理工科研究者的时间，正在被两件事消耗：
> 一是"写代码"——从零开始写数据处理、画图、跑实验的脚本；
> 二是"格式化工作"——排版、引用、期刊格式套用。
> 这两件事 AI 现在都可以做到 80% 以上的质量。
> **把这部分时间还给实验设计和科学发现，才是理工科研究者的正确打开方式。**

## 四、实战案例：材料科学实验数据分析

### 场景：XRD 数据批量处理与可视化

**传统方式**：手动整理 Excel → 逐个画图 → 调参数对齐格式（3-5 小时）

**AI 辅助方式**：

**Step 1**：打开 Cursor，描述需求：
```
我有 20 个 XRD 数据文件（.txt格式），
每个文件有两列：2theta 和 intensity
需要：
1. 批量读取所有文件
2. 在一张图上叠加显示所有曲线，不同颜色
3. 标注各曲线的峰位（找极值）
4. 输出论文级别的图（300dpi，Times New Roman字体）
5. 同时输出每个样品的峰位数据汇总表
```

**Cursor 生成完整 Python 脚本**（约 60 行），你只需要：
- 修改文件路径
- 确认峰位识别逻辑是否符合你的实验情况
- 调整图的样式细节

**节省时间**：从 3-5 小时 → 30 分钟

---

### 场景：实验方案设计

**Prompt 实际输入**：
```
我在研究[钙钛矿太阳能电池]的[界面钝化]问题，
目前常用的方法有：[列出已知方法]

请帮我：
1. 分析每种方法的优势和局限
2. 推荐 3 种我可以尝试的新钝化策略
3. 对于每种策略，列出需要验证的关键实验参数
```

**AI 输出**：系统性的方案对比表 + 参数清单

**你的判断**：结合实验室设备条件、材料获取难度、与导师方向的契合度，做最终决策

---

### 场景：Discussion 写作

**Prompt 实际输入**：
```
我的实验结果如下：
- 样品 A 的效率为 22.1%，比对照组提高 3.2%
- EIS 测试显示界面电阻降低 40%
- PL 测试显示缺陷态密度下降

请帮我起草 Discussion 部分，要求：
1. 解释为什么效率提高（结合我的钝化机制）
2. 将我的结果与文献[列出 3 篇]的结果比较
3. 分析本研究的局限性
4. 提出未来研究方向
```

## 五、代码相关的 AI 使用技巧

### 高效 Debug

```
以下是我的 Python 代码和报错信息：

代码：[粘贴]
报错：[粘贴完整 traceback]
数据格式：[描述输入数据的结构]

请：
1. 解释报错原因
2. 给出修改后的代码
3. 说明是否有其他潜在 bug
```

### 代码 Review（提交前）

```
以下是我的实验核心代码：[粘贴]
请检查：
1. 计算逻辑是否有错误
2. 是否有内存效率问题（数据量大时）
3. 结果的可重复性（随机数种子等）
4. 是否有更简洁的实现方式
```

### 理工科英文写作专项 Prompt

```
# Methods 部分
以下是我实验步骤的中文描述：[粘贴]
请转写为学术英文的 Methods 部分，
要求：被动语态为主，过去时，精确描述参数

# 图表说明（Figure Caption）
这张图展示了[描述图的内容和关键信息]
请写一段 Figure Caption，50-100词，
要求：先描述图的内容，再指出关键发现
```

## 六、理工科方向期刊推荐策略

**Prompt**：
```
我的论文：
- 方向：[具体方向，如：钙钛矿太阳能电池界面工程]
- 主要贡献：[效率提升幅度 / 新机制 / 新方法]
- 数据量：[实验数量、表征手段]

推荐 5 个适合投稿的期刊，注明：
- 期刊对该方向的接收偏好
- 当前影响因子区间
- 审稿周期
- 是否需要付 APC（开放获取费）
```

**理工科常见高质量期刊分类**：
- 综合性：Nature / Science / PNAS（顶刊）
- 材料：Advanced Materials / ACS Nano / Nano Energy
- 能源：Joule / Energy & Environmental Science
- 化学：JACS / Angewandte Chemie
- 工程：IEEE 系列 / Applied Physics Letters
