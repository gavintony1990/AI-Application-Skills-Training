# AI 赋能科研提效课程体系

## 课程定位

面向研究生与博士生，将论文写作与投稿流程工程化，
用 AI 系统性替代机械工作、辅助分析工作，显著提升研究效率。

---

## 目录结构

```
001_方法理论总述/          ← 所有专业通用的核心框架与工具体系
002_法学与社会科学/        ← 法学、政治学、社会学方向
003_理工科/               ← 工程、物理、化学、材料方向
004_生命科学与医学/        ← 生物、医学、药学方向
005_计算机与人工智能/      ← CS、AI、数据科学方向
006_人文学科/             ← 历史、文学、哲学、语言学方向
007_经济与管理学/         ← 经济学、管理学、金融方向
```

---

## 使用建议

- **第一次讲课**：先讲 001，再选对应专业方向的模块
- **单独使用**：每个专业目录均可独立使用，内含通用方法的简版回顾
- **实战演示**：每个专业目录均包含 1 个完整实战案例，可现场演示

---

## 核心理念

> 不要用博士的大脑做助理的工作
> 把流程设计好，让 AI 替你跑
