# AI 商业与创业讲座系列

## 系列定位

面向创业者、产品经理、企业管理者、投资人，
从商业逻辑、产品策略、组织变革、投资机会多个维度，
系统拆解 AI 时代的商业机会与创业方法论。

## 使用方式

- **单场讲座**：选取一个主题，配合对应文件的 PPT 提纲独立使用
- **系列课程**：L01→L02→L05→L06 构成创业者完整学习路径
- **企业内训**：L03 + L01 组合，适合企业高管团队
- **投资机构**：L04 + L02 组合，适合 VC/PE 行业研究
